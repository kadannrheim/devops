
# oauth2-proxy
OAuth2 Proxy - это инструмент с открытым исходным кодом, который обеспечивает аутентификацию и авторизацию для приложений, используя протоколы OAuth2 и OpenID Connect. Он обеспечивает аутентификацию через внешний источник, такой как Google, Github, Keycloak и другие, и может быть использован для защиты внутренних приложений.

# Ingress 
Ingressв Kubernetes (K8s) — это API‑объект, который управляет внешним доступом к сервисам внутри кластера (обычно по HTTP/HTTPS) через правила маршрутизации, исполняемые Ingress Controller.
